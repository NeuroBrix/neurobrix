# Getting Started

Get NeuroBrix up and running in under 5 minutes.

## Prerequisites

- Python 3.10+
- PyTorch 2.0+ with CUDA support
- NVIDIA GPU (V100, A100, RTX series)

1. **[Install](installation.md)** — Install NeuroBrix from PyPI
2. **[Quick Start](quickstart.md)** — Run your first model
3. **[Model Hub](hub.md)** — Browse and import models
